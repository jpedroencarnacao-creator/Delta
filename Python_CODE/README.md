# Python_CODE

Scripts Python do projeto.

## Ambiente

Dependencias:

```powershell
python -m pip install -r Python_CODE/requirements.txt
```

## Codigos antigos

- `codigos_antigos/GUI_V2.py`: interface NiceGUI para Windows/PC.
- `codigos_antigos/GUI_PI4B.py`: interface NiceGUI com ajustes para Raspberry Pi.
- `codigos_antigos/GUI_V1.py`: versao anterior/teste da interface.
- `codigos_antigos/Serial_Comunication_test.py`: testes de comunicacao serie.

## VS Code

Os ficheiros devem usar extensao `.py` para a extensao Python/Pylance reconhecer imports, linting, autocomplete e debug.
